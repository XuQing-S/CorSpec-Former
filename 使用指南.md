# 使用指南

本文档说明如何使用当前项目中的深度学习代码完成 LIBS 金属牌号 6 分类训练、测试和结果查看。

## 1. 前置条件

**数据准备**：

1. 确认已经存在预处理后的张量目录，例如：
   - `data/processed/balanced`
   - `data/processed/noise_shift/1`
   - `data/processed/noise_shift/2`
2. 目录中应至少包含：
   - `train_x.npy`、`val_x.npy`、`test_x.npy`
   - `train_y_metal.npy`、`val_y_metal.npy`、`test_y_metal.npy`
   - `label_mapping.json`

**环境准备**：

1. 先安装基础依赖：
   - `pip install -r requirements.txt`
2. 再按本机 CUDA 版本安装 PyTorch。示例：
   - `pip install torch torchvision --index-url https://download.pytorch.org/whl/cu128`
3. 如果只使用 CPU，可安装 CPU 版本 PyTorch。

当前训练代码入口：

- 训练：`scr/dl/train.py`
- 测试：`scr/dl/test.py`
- 数据集：`scr/dl/dataset.py`
- 模型：`scr/dl/models/`
- 默认配置：`configs/default/`

## 2. 使用配置文件训练

推荐优先使用配置文件启动训练。当前默认配置包括：

1. `configs/default/fnn.json`
2. `configs/default/cnn.json`
3. `configs/default/lstm.json`
4. `configs/default/resnet.json`

### 训练 FNN

```bash
python scr/dl/train.py --config configs/default/fnn.json
```

### 训练 1D CNN

```bash
python scr/dl/train.py --config configs/default/cnn.json
```

### 训练 LSTM

```bash
python scr/dl/train.py --config configs/default/lstm.json
```

### 训练 1D ResNet

```bash
python scr/dl/train.py --config configs/default/resnet.json
```

## 3. 命令行覆盖配置

配置文件只需要写需要调整的参数。未写出的参数使用 `train.py` 内置默认值。

命令行参数优先级最高，可以临时覆盖配置文件。

### 修改运行名称

```bash
python scr/dl/train.py --config configs/default/cnn.json --run-name cnn_try_01
```

### 修改学习率

```bash
python scr/dl/train.py --config configs/default/cnn.json --learning-rate 0.0005
```

### 修改训练轮数和 batch size

```bash
python scr/dl/train.py --config configs/default/resnet.json --epochs 150 --batch-size 32
```

### 切换数据集

```bash
python scr/dl/train.py --config configs/default/cnn.json --processed-root data/processed/noise_shift/1
```

### 关闭数据分布类可视化

```bash
python scr/dl/train.py --config configs/default/cnn.json --no-data-figures
```

### 保存 t-SNE 图

t-SNE 较慢，建议只在最终实验或小样本可视化时开启。

```bash
python scr/dl/train.py --config configs/default/cnn.json --save-tsne --visualize-max-samples 1000
```

## 4. 修改模型参数

模型结构参数写在配置文件的 `model_params` 中。

### FNN 常用参数

```json
{
  "model": "fnn",
  "model_params": {
    "input_length": 8192,
    "hidden_dims": [1024, 256, 64],
    "dropout": 0.3
  }
}
```

可调整：

1. `hidden_dims`：全连接层宽度。
2. `dropout`：防止过拟合。

### CNN 常用参数

```json
{
  "model": "cnn",
  "model_params": {
    "input_channels": 1,
    "channels": [32, 64, 128, 256],
    "kernel_sizes": [7, 5, 5, 3],
    "pool_sizes": [2, 2, 2, null],
    "dropout": 0.3
  }
}
```

可调整：

1. `channels`：每层卷积通道数。
2. `kernel_sizes`：卷积核大小。
3. `pool_sizes`：池化比例，`null` 表示该层不池化。
4. `dropout`：分类头 dropout。

### LSTM 常用参数

```json
{
  "model": "lstm",
  "model_params": {
    "conv_channels": 64,
    "conv_stride": 4,
    "hidden_size": 128,
    "num_layers": 1,
    "bidirectional": true,
    "pooling": "mean",
    "dropout": 0.3
  }
}
```

可调整：

1. `conv_stride`：前置卷积降采样比例。
2. `hidden_size`：LSTM 隐状态维度。
3. `num_layers`：LSTM 层数。
4. `bidirectional`：是否使用双向 LSTM。
5. `pooling`：序列聚合方式，可选 `mean` 或 `last`。

### ResNet 常用参数

```json
{
  "model": "resnet",
  "model_params": {
    "base_channels": 32,
    "blocks_per_stage": [2, 2, 2, 2],
    "stem_kernel_size": 7,
    "stem_stride": 2,
    "dropout": 0.3
  }
}
```

可调整：

1. `base_channels`：初始通道数。
2. `blocks_per_stage`：每个 stage 的残差块数量。
3. `stem_kernel_size`：首层卷积核大小。
4. `stem_stride`：首层降采样比例。
5. `dropout`：分类头 dropout。

## 5. 输出结果

训练结果默认保存到：

```text
experiments/dl/<run_name>/
```

主要文件包括：

1. `config.json`：本次实验配置。
2. `metrics.json`：验证集最佳指标和测试集指标。
3. `history.csv`：每轮训练损失、验证损失、验证指标、学习率和耗时。
4. `classification_report.txt`：测试集分类报告。
5. `report.md`：实验摘要。
6. `checkpoints/best_model.pt`：验证集 Macro-F1 最佳模型。
7. `checkpoints/last_model.pt`：最后一轮模型。
8. `predictions/metal_<model>_test.csv`：测试集逐样本预测。
9. `confusion_matrix.csv`：测试集混淆矩阵。
10. `figures/`：训练曲线、混淆矩阵、PCA 分布图、类别平均光谱等。

## 6. 独立测试模型

训练完成后，可以用 `scr/dl/test.py` 重新评估 checkpoint。

### 基本测试

```bash
python scr/dl/test.py --checkpoint experiments/dl/cnn_baseline/checkpoints/best_model.pt
```

### 指定输出目录

```bash
python scr/dl/test.py \
  --checkpoint experiments/dl/cnn_baseline/checkpoints/best_model.pt \
  --output-dir experiments/dl/cnn_baseline/test_eval
```

### 保存测试集 PCA embedding

```bash
python scr/dl/test.py \
  --checkpoint experiments/dl/cnn_baseline/checkpoints/best_model.pt \
  --save-embedding
```

测试输出默认保存到：

```text
experiments/dl/<run_name>/test_eval/
```

主要包括：

1. `test_metrics.json`
2. `classification_report.txt`
3. `test_report.md`
4. `predictions/`
5. `confusion_matrix.csv`
6. `figures/`

## 7. 推荐实验顺序

建议按以下顺序实验：

1. 先训练 `cnn`，作为主要深度学习基线。
2. 再训练 `fnn`，判断直接展平光谱的效果。
3. 再训练 `resnet`，判断更强卷积模型是否提升。
4. 最后训练 `lstm`，作为序列依赖对比模型。

推荐命令：

```bash
python scr/dl/train.py --config configs/default/cnn.json
python scr/dl/train.py --config configs/default/fnn.json
python scr/dl/train.py --config configs/default/resnet.json
python scr/dl/train.py --config configs/default/lstm.json
```

如果要测试噪声偏移数据集：

```bash
python scr/dl/train.py --config configs/default/cnn.json --processed-root data/processed/noise_shift/1 --run-name cnn_noise_shift_1
```

## 8. 常见问题

### 运行时报 `ModuleNotFoundError: No module named 'torch'`

说明当前环境没有安装 PyTorch。需要根据 CUDA 或 CPU 环境安装 PyTorch。

### 训练很慢

可尝试：

1. 使用 GPU。
2. 降低 `batch_size` 或模型通道数。
3. 关闭 t-SNE：不要使用 `--save-tsne`。
4. 关闭数据分布图：使用 `--no-data-figures`。

### 显存不足

可尝试：

1. 降低 `--batch-size`。
2. 减小 CNN 的 `channels`。
3. 减小 ResNet 的 `base_channels` 或 `blocks_per_stage`。
4. 减小 LSTM 的 `hidden_size` 或 `num_layers`。

### 配置文件参数没有生效

检查：

1. 是否使用了正确的 `--config` 路径。
2. 参数字段名是否和 `train.py` 中一致。
3. 是否又在命令行中传入了同名参数，命令行参数会覆盖配置文件。

### 输出目录被覆盖

如果多次使用相同 `run_name`，输出会写到同一目录。建议每次实验使用不同运行名：

```bash
python scr/dl/train.py --config configs/default/cnn.json --run-name cnn_baseline_02
```
